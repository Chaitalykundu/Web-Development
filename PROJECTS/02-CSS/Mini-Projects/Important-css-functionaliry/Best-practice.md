# CSS shorthand

## don't use

```css
.home {
  background-color: #000;
  background-image: url(image/bg.jpg);
  background-repeat: no-repeat;
  background-position: left top;
}
```

## Use

```css
.home {
  background: #000 url(image/bg.jpg) no-repeat left top;
}
```

&nbsp;

&nbsp;

# Don't prefix property values with a leading 0

## Don't use

```css
.selector {
  padding: 0.5px;
}
```

## Use

```css
.selector {
  padding: .5px;
}
```

&nbsp;

&nbsp;

# Avoid specifying units for 0 values

## Don't use

```css
.selector {
  padding: 0px;
}
```

## Use

```css
.selector {
  padding: 0;
}
```

&nbsp;

# Declaration property in order

```css
.selector {
  /* positioning */
  position: absolute;
  top: 0;
  right: 0;

  /* box-model */
  display: block;
  width: 50%;
  max-width: 500px;
  padding: 23px;
  border-radius: 1px;

  /* typography */
  font-size: 16px;
  font-family: sans-serif;

  /* visual */
  background: var(--bg-color);
  border: 1px solid black;
  filter: var(--filter);
}
```

# Use meaningful class names with lowercase and dashes

## don't use

```css
.dltbtn {
}

.msg {
}
```

## use

```css
.delete-button {
}

.message {
}
```
