# `<br>` tag

- The `<br>` tag is used to create a single line break in a text.

- The `<br>` tag is useful for writing addresses or poems.

&nbsp;

&nbsp;

# Syntax

```html
<Tag_name>Content</Tag_name> <br />
```

&nbsp;

# Example

```html
<p>This is paragraph</p>
<br />
<p>This is second paragraph</p>
```
