# Definition and Usage

The `<legend>` tag defines a caption for the `<fieldset>` element.

The `<legend>` element must be the **first child** of `<fieldset>` element.

&nbsp;

&nbsp;

# Syntax

```html
<legend>Caption.....</legend>
```

# Default CSS Settings

```css
legend {
  display: block;
  padding-left: 2px;
  padding-right: 2px;
  border: none;
}
```
