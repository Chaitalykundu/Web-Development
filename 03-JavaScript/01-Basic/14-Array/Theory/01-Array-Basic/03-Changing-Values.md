# Change the value(s)

Arrays are `mutable`. Means the values of an array can be changed.

&nbsp;

# Syntax

```js
array_name[index_no_whose_value_to_be_changed] = value;
```

&nbsp;

&nbsp;

# Example

```js
const cars = ["Saab", "Volvo", "BMW"];
cars[0] = "Opel";
```
