## Loops in js

- for loop
- for in loop
- for of loop
- while loop
- do-while loop

&nbsp;

## for loop

- Syntax :

  ```
  for(statement1 ; statement2 ; statement3){
      code to be executed;
  }
  ```

&nbsp;

## for-in loop

- We use for-in loop in object and array
- Syntax :

  ```
    for( let key in object_name)
      console.log(object_name[key]);
  ```

  &nbsp;

  ## for-of loop

- We use for-in loop in object and array
- Syntax :

  ```
    for( variable of object_name)
      code
  ```
