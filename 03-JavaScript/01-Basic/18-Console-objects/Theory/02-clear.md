# Description

The `clear()` method clears the console.

The `clear()` method also write "Console was cleared" in the console.

&nbsp;

&nbsp;

# Syntax

```js
console.clear();
```

&nbsp;

&nbsp;

# Examples

```js
let x = 2,
  y = 3;

console.log(x + y);

console.clear();
```
