# Non-Primitive / Object / Reference datatypes

The Object Datatypes are :

1. An object
2. An array
3. A Function

&nbsp;

| Type     | typeof return value                  |
| -------- | ------------------------------------ |
| object   | "object"                             |
| array    | "object"                             |
| function | "function" ( this is objectfunction) |
|          |                                      |

&nbsp;

&nbsp;

# Object

```js
const car = { type: "Fiat", model: "500", color: "white" };
```

&nbsp;

&nbsp;

# Array

```js
const cars = ["Saab", "Volvo", "BMW"];
```
