# trim()

- to remove the spaces of string

## Syntax :

```js
const firstname = "            Chaitaly           ";
const lastName = "Kundu";
console.log(firstname, lastName); //            Chaitaly            Kundu
console.log(firstname.trim(), lastName); //Chaitaly Kundu
```
