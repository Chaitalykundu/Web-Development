# break statement

The `break` statement **"jumps out"** of a loop.

The break statement can be used to jump out of a loop or a switch.

&nbsp;

# Example

```js
for (let i = 0; i < 10; i++) {
  if (i === 3) {
    break;
  }
  text += "The number is " + i + "<br>"; // 0 1 2
}
```
