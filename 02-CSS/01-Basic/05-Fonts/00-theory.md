CSS **Font** property is used to **control the look of texts**.

&nbsp;

By the use of CSS font property you can change the **text size, color, style** and more.

&nbsp;

CSS **font** properties are :

- font-style
- font-variant
- font-weight
- font-size
- line-height
- font-family
- color
- font-stretch
