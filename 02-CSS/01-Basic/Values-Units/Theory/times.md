# CSS units: Time

Some `animation` properties require values to express in time.

&nbsp;

| Unit | Explanation                                               |
| ---- | --------------------------------------------------------- |
| s    | duration of time in seconds.                              |
| ms   | duration of time in milliseconds. 1ms = 1/100 of a second |
