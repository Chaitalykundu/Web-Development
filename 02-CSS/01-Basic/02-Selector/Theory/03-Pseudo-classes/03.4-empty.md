# Chapter Summary

- The `:empty` selector matches every element that has no children (including text nodes).

&nbsp;

&nbsp;

# Definition and Usage

- The `:empty` selector matches every element that has no children (including text nodes).

&nbsp;

&nbsp;

# CSS Syntax

```css
:empty {
  css declarations;
}
```

&nbsp;

&nbsp;

# Example

Specify a background color for empty `<p>` elements:

```css
p:empty {
  width: 100px;
  height: 20px;
  background-color: #ff0000;
}
```
