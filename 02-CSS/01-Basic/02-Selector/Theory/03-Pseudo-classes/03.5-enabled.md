# Definition and Usage

- The `:enabled` selector matches every enabled element (mostly used on **form** elements).

&nbsp;

&nbsp;

# CSS Syntax

```css
:enabled {
  css declarations;
}
```

&nbsp;

&nbsp;

# Example

Set a background color for all enabled `<input>` elements:

```css
input:enabled {
  background: #ffff00;
}
```
