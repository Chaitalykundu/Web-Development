# Definition and Usage

- The `:checked` selector matches every checked `<input>` element (only for **_radio buttons and checkboxes_**) and `<option>` element.

&nbsp;

&nbsp;

# CSS Syntax

```css
:checked {
  css declarations;
}
```

&nbsp;

&nbsp;

# Example

Set the height and width for all checked `<option>` elements:

```css
option:checked {
  height: 100px;
  width: 100px;
}
```
