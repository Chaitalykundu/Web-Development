# Definition and Usage

- The `::marker` selector selects the marker of a list item.

* This selector works on any element set to `display:list-item`.

&nbsp;

&nbsp;

# CSS Syntax

```css
::marker {
  css declarations;
}
```

&nbsp;

&nbsp;

# Example

- Style the markers of list items:

```css
::marker {
  color: red;
  font-size: 23px;
}
```
