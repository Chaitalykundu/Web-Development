## Datatype

Datatype is the type of the data stored in a variable.

There are two types of datatypes in js.
_ Primitive
_ Non-primitive

&nbsp;

## Primitive datatypes

- Primitive datatypes are the set of basic datatypes in javascript
- There are 7 types of datatypes

  ```
  NNSSBBU

  N = number
  N = null
  S = string
  S = symbol
  B = boolean
  B = BigInt
  U = undefined
  ```
