## if condition

- Syntax :

```
    if(condition){
        exp1;
    }
```

&nbsp;

## if-else condition

- Syntax :

```
    if(condition){
        exp1;
    }
    else{
        exp2;
    }
```

&nbsp;

## if - else if condition

- Syntax :

```
    if(condition){
        exp1;
    }
    else if(condition){
        exp2;
    }
    else{
        exp3;
    }
```

&nbsp;

## Ternary operation

- Syntax :

```
    condition ? exp1 : exp2
```

&nbsp;
